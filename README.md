## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
# to use app with docker (app, mysql, redis)
$ npm run dev
# production mode
$ npm run start:prod

# launch container mysql and redis
$ npm run dockerEnvUp
# stop container mysql and redis
$ npm run dockerEnvStop
# remove container mysql and redis
$ npm run dockerEnvRm
# stop and remove container mysql and redis
$ npm run dockerEnvDown

```
